# 内田クレペリン検査 練習サイト

内田クレペリン検査（一桁の数字を隣同士で足し算し、答えの一の位を記入していく作業検査）を
ブラウザ上で練習できる、サーバー不要の静的Webサイトです。

## 特徴

- **フロントエンド完結**：HTML / CSS / Vanilla JavaScript のみで動作。ユーザー登録やサーバー通信は一切なし。
- **2つのモード**
  - 本番モード：前半15分 → 休憩1分 → 後半15分（実際の検査に近い構成）
  - 練習モード：休憩なしの5分間（お試し用）
- **PC / スマホ両対応**
  - PC：キーボードの数字キーで入力、自動的に次のマスへフォーカス移動
  - スマホ・タブレット：画面下部に固定表示された0〜9のテンキーボタンで入力
- **1分ごとの自動改行＋通知音**：1分経過するたびに音が鳴り、自動的に次の行の先頭に移動します（本物の検査と同じ仕様）。
- **結果画面**
  - 作業曲線（折れ線グラフ、前半・後半で色分け）
  - スタッツ（平均処理数、正答率、最高／最低処理数など）
  - 結果に応じた簡易フィードバックコメント

## 使い方

1. `index.html` をブラウザで直接開くだけで利用できます（ローカルサーバー不要）。
2. スタート画面でモード（本番／練習）と通知音のON/OFFを選択し、「検査を開始する」を押します。
3. 画面に表示された数字を隣同士で足し算し、その答えの一の位の数字を入力していきます。
   - 例：`3 5 8 2` → 3+5=8 → 5+8=13→「3」→ 8+2=10→「0」
4. 1分経過すると自動的に次の行へ移動します。最後の行まで終えると結果画面が表示されます。

## ファイル構成

```
kraepelin/
├── index.html          # 全画面（スタート／検査／休憩／結果）を1ページで構成
├── css/
│   └── style.css       # 落ち着いた学校向けデザイン、レスポンシブ対応
├── js/
│   ├── sound.js         # Web Audio APIによる通知音生成
│   ├── test-engine.js   # 数字生成・採点・タイマー用状態管理ロジック
│   ├── chart-render.js  # 作業曲線グラフ描画（Chart.js使用）
│   └── main.js          # 画面制御・DOM操作・イベントハンドリング
└── README.md
```

## 技術メモ

- グラフ描画には [Chart.js](https://www.chartjs.org/)（CDN経由）を利用しています。
- 通知音は外部音声ファイルを使わず、Web Audio APIでビープ音を生成しているため、
  余分なアセットファイルなしで動作します。
- すべての採点・集計処理はブラウザ内のJavaScriptで完結しており、サーバーやデータベースは不要です。
